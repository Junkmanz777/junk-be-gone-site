# Junk Be Gone Clovis Website

Official website for **Junk Be Gone**, a local junk removal service based in Clovis, New Mexico.

The site includes information about junk removal services, service area, customer reviews, TikTok work videos, and a free-stuff request page for useful items found during pickups.

## Business

**Junk Be Gone**  
Clovis, New Mexico  
Phone: (575) 530-0144  
Website: https://www.junkbegoneclovis.com

## Features

- Junk removal service information
- 20-mile service area map around Clovis, NM
- Click-to-call buttons
- Google reviews link
- TikTok video section
- Free-stuff request form
- Mobile-friendly design
- Render-hosted Flask website

## Technology

This site is built with:

- Python
- Flask
- HTML
- CSS
- Leaflet / OpenStreetMap
- FormSubmit for email form submissions
